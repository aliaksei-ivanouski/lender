# TASK-6 — Wave 3: Attendees & Confirmation Email

**Status:** in_progress
**Created:** 2026-06-18
**User Stories:** US-301 (attendee registration + list), US-302 (confirmation email), US-203 (event detail page as the registration home)

## Goal
Let authenticated users register interest/attendance for an event (an attendee list), and email them a confirmation ("you're on the list") when added. Build the event detail page (Events/Show.vue, currently a stub at the existing /events/{event} route) into the home for the Register/Unregister control + attendee list, showing the event's images, human-readable address, and event-local date/time. Mail driver is `log` (emails written to storage/logs). Reminder emails (3-day / 24-hour) are Wave 4 (TASK-7), out of scope here.
